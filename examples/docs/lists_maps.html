Lists and Maps

List

Lists must contain data of the same type. The type must be specified in < > when creating the list. [] notation can be used to access and set the elements in a List. Lists also have the 'size' property which can be used to Lists are passed by reference.


Lists can be sliced to using [] notation with the starting and ending indices separated by ':'. See the code snippet below for an example.


The '++' operator will concatenate Lists and elements and return a new List object. Note: Using '++' will create a new List and copy over any elements in the concatenated Lists. This can be slow if the Lists are large. If you just want to append to a List, using [] notation to set the next element in the List will be more efficient. 

                
    my_list: List = List()

    print(my_list.size) //0

    my_list[0] = 2
    my_list[1] = 11
    my_list[2] = -9

    foreach i: int in my_list {
        print(i)
    }

    //211-19

    //make a new list with the first two elements of 'my_list'
    sliced_list = my_list[0:2]
    foreach i: int in sliced_list {
        print(i)
    }

    //211

    other_list := 1 ++ 3 //concatenate two elements, creating a new List
    other_list = other_list ++ 4 //append 4
    other_list = 0 ++ other_list //prepend 0
    other_list = other_list[0:2] ++ 2 ++ other_list[2:4] //insert 2 at index 2

    foreach i: int in other_list {
        print(i)
    }

    //01234

Map

Each entry in a Map is a key-value pair. The key is a string type and the value type is specified at creation of the Map using < >. Use the properties 'keys' and 'values' to get a List of all keys and values, respectively, in the Map. 

                
    my_map: Map = Map()

    print(my_map.size) //0

    my_map["dog"] = 32.2
    my_map["cat"] = -2.6

    foreach k: string in my_map.keys {
        print(k + " " + my_map[k])
    }

    //dog 32.2
    //cat -2.6
    //Note: order of key/value pairs may vary
